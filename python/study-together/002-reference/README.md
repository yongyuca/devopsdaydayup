# 组团学Python第2期班会-参考资料

同学们好！今天开始，我们将以不定期开班会的形式和大家一起系统性的学习Python。

我们的学习方式是，我先抛些我的学习心得，然后大家可以参与讨论，之后一起做一些练习题巩固知识，加强肌肉记忆。因为我也算是初学，一定有我理解不正确的地方，欢迎大家随时指正。

有研究表明，学习效率最低的方式是阅读，其次是看视频，然后是讨论，效率最高的是以教为学。

网上自学Python的资料非常多，但我觉得，我们的学习群最大的优势就是给大家一个参与讨论甚至分享教学的机会，并且可以相互拿小鞭子鼓励监督。回答别人的一个问题或者提出自己的一个问题，都是在主动创造和这个知识点的接触机会，你主动参与的越多，你学习的就越多，你也就能记得更久一些，让这些知识不再只是你头脑里的匆匆过客。

希望大家不要错过这个机会，利用这几个月的时间，把这门当下最热门的语言拿下，为你的职业生涯增添更多的可能。

话不多说，我们切入正题吧。

哦，对了，开始之前，我先列一下我的学习资料源，以下学习材料这一个月我大概都翻了一下，感觉不错，之后会挑一些印象深刻的和大家分享。

以下排名分先后，鼓励大家有时间也去读一下，毕竟一手知识一定比我这二手转述来的更完整。我对自己的定位只是抛砖引玉，督促你学习，真正有价值的知识还得回到源头自己读。

1.看漫画学Python（书分享在群里了，这本书的章节结构划分的很好，我们基本就按照它的结构系统的学）
2.廖雪峰的Python教程（文字生动有趣，也非常适合初学者入门，我会结合我们的进度争取也每章节都能学习到）
3.极客时间 - 零基础学Python(2023版) 尹会生 （这是付费视频课程，内容还比较系统全面，留言区也很多有价值的讨论，以后也会时不时的和大家分享）
4.Python Crash Course, 3rd Edition （电子书，经典入门教材，系统详细，还有一些有趣的项目，到时候会带着大家一起做）
5.Python工匠（电子书，对于有一定经验的同学非常推荐，书里有很多非常实用的Best Practice，能够让你从初级走向中级）
6.极客时间 - Python核心技术与实战 （付费专栏，一些独到的技术观点，到时候也会时不时和大家分享）
7.极客时间 - Python自动化办公实战 尹会生（付费视频，之前有同学说过想通过Python来处理一些Excel文件，之后有机会我们可以一起做）
8.极客时间 - 数据结构与算法之美 王争（这是付费专栏，别忘了咱年底的两个目标其中第一，就是会做中级算法题，为你的下一份工作的面试做好准备）
9.Learn Python3 the Hard Way（书里除了有丰富的练习题，最主要的是我很喜欢这本书的理念：只要将问题拆分成小的练习和课程，你就可以学会任何东西；只要集中精力慢慢提高，享受学习的过程，不管你最终学到什么程度，你都会从中获益的。）
10.100 Days of Code：The Complete Python Pro Bootcamp-Dr. Angela Yu（Udemy高分课程，因为昨天才买的课，所以先排在最后，看了一下大纲，感觉从零开始学Python，很详细很具体，推荐没基础的同学可以参加）

以上资料是我今后分享的信息源，如果你有更好的资料，欢迎推荐分享。

那咱就开始吧！

首先我们简单的讨论一下为啥学Python，咱之前也讨论过，非常推荐大家想清楚这个问题，然后给自己定好目标，这样学习起来才会事半功倍。我之前说过，我的最主要目标就是工作面试，这可是硬指标，一些我想去的公司必须考算法，不会就是进不了。

这两天其实我还在想着另外一个事，就是如果说，运气好，这次Python真就学成了，那我想做一个小应用，具体是什么没想好。这个灵感主要来源于《黑客与画家》这本书，它说，黑客和画家的共同点，就是他们的工作本身就是在创造财富，黑客敲出的每一行代码，画家画出的每一笔就是财富本身。而所谓的财富，就是能给社会带来的多余收益。

从这个角度看，学好一门编程，就能直接创造财富！
